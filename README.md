# HelathCare Website

## Description

This is a healthcare website that allows patients to register, schedule appointments with doctors, veiw health history, and get health advice.
Doctors can manage patients and appointments, while administrators can manage useres and view statistics

# Features

Register and login for patients, doctors and admin.
Appontment scheduling and management.
Health history viewing.

# Technologies used

HTML, Tailwind css, TypeScript and React.js.
Uses MongoDB for data persistence.
